# Product Requirements

## Vision

A GPS-based alarm that wakes commuters before reaching their destination stop.

---

## Primary User

Student commuter who may sleep during bus travel.

---

## Must-Have Features

1. Distance-based alarm creation
2. Live map view (MapLibre)
3. Foreground location tracking
4. Alarm engine (sound + vibration)
5. Clean Compose UI

---

## Constraints

* Solo developer project
* Android only
* No backend/server
* Client-side logic only
* Battery drain must stay under 10%/hour

---

## Success Metrics

* Alarm triggers reliably
* Wake-up success rate >95%
* Crash-free sessions >99%
* Works with screen off (Phase 2)

---

## Definition of Done

* Foreground alarm reliable
* Background alarm reliable
* Field tested in real commute
* Battery within limits
